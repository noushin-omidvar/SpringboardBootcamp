DOM and DOM events

2nd week learning summary, Dec 18, 2022

1. DOM

Document Object Model (DOM) is the programming interface for HTML and a representation of it that can be accessed using JavaScript.

Using JavaScript and through DOM we can manipulate HTML.

The document object has properties and functions that can help us to access HTML elements such as:

  • getElementById(): finds the first matching id
  • getElementByTagName(): returns a list of all of the elements that match the string passed to the function
  • getElementByClassName(): returns a list of all of the elements that have a class attribute, which matches the string passed to the function
  • querySelector(): returns the first element that matches the CSS selector passed to the function.
  • querySelectorALL(): returns all the elements that matches the CSS selector passed to the function

2. DOM manipulation

Values and attributes of elements in DOM can be modified.

Text can be modified by accessing:

  • innerHTML
  • innerText
  • textContent
Any inline CSS properties on an element can be accessed using the style property.

In order to access an attribute on an element, we can use the getAttribute method and to set an attribute on an element, we can use the setAttribute method.

To manipulate the class attribute in JavaScript, we can use:

  • setAttribute(“class”) - this will override the class
  • className - this will give you a string representation of the class
  • classList - this will give you an array-like object to add, remove or toggle classes

To create an HTML element, we can use the createElement function and pass in the name of the element

After creating an element we can append/ prepend it as child to a parent element

If we want to remove elements in the DOM, we can use remove method.

We can access siblings and children elements using relevant attributes like children, firstElementChild, lastElementChild, previousElemnetChild, previousElementSibling and nextElementSibling.

3. DOM Events

DOM events are “actions” that occur as a result of something the user or the browser does. We can write JavaScript codes that listens for events - and react accordingly! One optintion is to use AddEventListener. Inside of the callback to addEventListener, we get access to a special object as a parameter - the event object.
